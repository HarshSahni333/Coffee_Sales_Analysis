## Project Overview

This project showcases an interactive Coffee Sales Analytics Dashboard developed in Microsoft Excel. The dashboard enables users to explore sales performance across products, countries, customer segments, and time periods through dynamic visualizations and filters.

## Project Objectives

* Analyze coffee sales performance over time
* Identify top-performing products and markets
* Explore customer purchasing behavior
* Create an interactive reporting solution for business users
